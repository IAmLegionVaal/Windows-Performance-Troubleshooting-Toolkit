# Windows Performance Troubleshooting Toolkit

A read-only PowerShell toolkit for Windows performance triage.

## Features

- CPU and memory snapshot
- Top processes by CPU and memory
- Disk free-space summary
- Uptime and OS context
- Recent performance-related event summary
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Performance_Troubleshooting_Toolkit.ps1
```

## Safety

Diagnostic-only. It reports performance context for support review.
